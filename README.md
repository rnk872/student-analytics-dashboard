# 🎓 Student Analytics Dashboard

A simple and interactive student dashboard built using pure HTML, CSS, and JavaScript. It allows users to view, search, and analyze student data in a clean and responsive interface.

---

## 🚀 Features

- 🔍 Real-time student search  
- 📊 Analytics slides (Total, Class-wise, Top scorer)  
- 👨‍🎓 120 auto-generated student profiles  
- ⚡ Fast and lightweight (no frameworks)  
- 📱 Responsive design  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
